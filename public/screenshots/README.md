# Screenshots

Drop real AlgoLens screenshots (or GIF/animated previews exported as images)
here. The site uses them automatically — until a file exists, a CSS mock is
shown in its place, so the site always looks complete.

| File         | Where it appears              | Suggested size      |
| ------------ | ----------------------------- | ------------------- |
| `hero.png`   | Hero product shot             | 1600 × 1000 (16:10) |
| `array.png`  | Demo → Array tab              | 1600 × 1000 (16:10) |
| `stack.png`  | Demo → Stack tab              | 1600 × 1000 (16:10) |
| `tree.png`   | Demo → Binary Tree tab        | 1600 × 1000 (16:10) |
| `graph.png`  | Demo → Graph tab              | 1600 × 1000 (16:10) |

Tips:
- Capture the AlgoLens window (or just the visualizer panel) on the dark theme.
- Keep a consistent aspect ratio (16:10) so nothing is cropped oddly.
- PNG for crisp UI; export GIFs as a representative still frame if needed.
